# Generate the SIS connect zip file

Generate the SIS connect zip file

## Usage

``` r
make_zip(csv_list = list(sis_files), zip_filename = "SIS_files.zip")
```

## Arguments

- csv_list:

  (list) List of SIS csv files you want to zip

- zip_filename:

  (character) Defaults to "SIS_files.zip"

## Value

Returns an SIS compliant zip file
