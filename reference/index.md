# Package index

## All functions

- [`check_occs()`](https://stevenpbachman.github.io/LCr/reference/check_occs.md)
  : Checks and flags GBIF occurrence data based on common coordinate
  issues
- [`clean_keys()`](https://stevenpbachman.github.io/LCr/reference/clean_keys.md)
  : Clean a keys output to ensure only accepted species from GBIF and
  WCVP are being analysed
- [`clean_occs()`](https://stevenpbachman.github.io/LCr/reference/clean_occs.md)
  : Clean GBIF occurrences based on user-selected flags and/or specific
  gbifIDs
- [`get_gbif_occs()`](https://stevenpbachman.github.io/LCr/reference/get_gbif_occs.md)
  : Download georeferenced occurrence data using GBIF name identifier
- [`get_name_keys()`](https://stevenpbachman.github.io/LCr/reference/get_name_keys.md)
  : Get accepted GBIF and POWO identifiers (keys) for taxon names to
  support data downloads
- [`get_native_range()`](https://stevenpbachman.github.io/LCr/reference/get_native_range.md)
  : Get native ranges for taxa
- [`get_occs_range()`](https://stevenpbachman.github.io/LCr/reference/get_occs_range.md)
  : Get ISO two-digit code from occurrence lat longs
- [`make_metrics()`](https://stevenpbachman.github.io/LCr/reference/make_metrics.md)
  : Generate metrics from a clean point occurrence file
- [`make_sis_csvs()`](https://stevenpbachman.github.io/LCr/reference/make_sis_csvs.md)
  : Generate all SIS connect csv files
- [`make_sis_occs()`](https://stevenpbachman.github.io/LCr/reference/make_sis_occs.md)
  : Convert cleaned gbif DWCA structure to IUCN SIS format
- [`make_zip()`](https://stevenpbachman.github.io/LCr/reference/make_zip.md)
  : Generate the SIS connect zip file
- [`map_species_batch()`](https://stevenpbachman.github.io/LCr/reference/map_species_batch.md)
  : Display a leaflet map to show the distribution of occurrence data
  for each species Batch version that iterates through species
- [`map_species_single()`](https://stevenpbachman.github.io/LCr/reference/map_species_single.md)
  : Display a leaflet map to show the distribution of occurrence data
  for each species
- [`search_name_gbif()`](https://stevenpbachman.github.io/LCr/reference/search_name_gbif.md)
  : Query taxon name against GBIF backbone
- [`search_name_powo()`](https://stevenpbachman.github.io/LCr/reference/search_name_powo.md)
  : Query taxon name against POWO (Plants of the World Online) backbone
- [`sis_allfields()`](https://stevenpbachman.github.io/LCr/reference/sis_allfields.md)
  : Generate the allfields.csv file
- [`sis_assessments()`](https://stevenpbachman.github.io/LCr/reference/sis_assessments.md)
  : Generate the assessments.csv file
- [`sis_countries()`](https://stevenpbachman.github.io/LCr/reference/sis_countries.md)
  : Generate the countries.csv file
- [`sis_credits()`](https://stevenpbachman.github.io/LCr/reference/sis_credits.md)
  : Generate the credits.csv file
- [`sis_habitats()`](https://stevenpbachman.github.io/LCr/reference/sis_habitats.md)
  : Generate the habitats.csv file
- [`sis_plantspecific()`](https://stevenpbachman.github.io/LCr/reference/sis_plantspecific.md)
  : Generate the plantspecific.csv file
- [`sis_references()`](https://stevenpbachman.github.io/LCr/reference/sis_references.md)
  : Generate the references.csv file
- [`sis_taxonomy()`](https://stevenpbachman.github.io/LCr/reference/sis_taxonomy.md)
  : Generate the taxonomy.csv file
