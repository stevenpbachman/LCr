# Generate the credits.csv file

Generate the credits.csv file

## Usage

``` r
sis_credits(
  unique_id,
  first_name = "your first name",
  second_name = "your second name",
  email = "your email",
  affiliation = "your affiliation"
)
```

## Arguments

- unique_id:

  (character) Unique identifier

- first_name:

  (character) First name of assessor

- second_name:

  (character) Second name of assessor

- email:

  (character) Email of assessor

- affiliation:

  (character) Name of institution or affiliation

## Value

Returns an SIS compliant csv file
