# Display a leaflet map to show the distribution of occurrence data for each species

Display a leaflet map to show the distribution of occurrence data for
each species

## Usage

``` r
map_species_single(data, species_range = NULL, show_flags = TRUE)
```

## Arguments

- data:

  A data frame containing occurrence data

- show_flags:

  Set to false if you just want to see the points map without flags

- native_ranges:

  A data frame containing occurrence data show_flags

## Value

Interactive leaflet map
